This LinkedIn automation bot is a Python-based tool designed to streamline the process of networking and lead generation on LinkedIn. Built using Selenium WebDriver, the bot automatically navigates LinkedIn profiles and performs actions such as visiting profiles, identifying the “Connect” button, and sending connection requests to targeted users.

The goal of the bot is to help professionals, freelancers, recruiters, and businesses grow their LinkedIn network efficiently without spending hours manually connecting with potential clients or partners.

The bot works by opening LinkedIn through an automated browser session, logging into a user account, and visiting a list of specified LinkedIn profile URLs. For each profile, it checks whether a “Connect” button is available and clicks it to send a connection request. If the button is not available (for example if the user already follows the profile or the request is pending), the bot skips that profile and moves on to the next.

To ensure safer automation, the bot can include randomized delays between actions to mimic human behavior and reduce the risk of LinkedIn detecting automated activity.

Key Uses:

Automating LinkedIn connection requests

Generating business leads

Growing a professional network faster

Helping freelancers find potential clients

Supporting recruiters in reaching candidates

Saving time on repetitive outreach tasks

This project demonstrates the use of Python automation, Selenium browser control, web scraping concepts, and workflow automation to solve a real-world networking problem.
